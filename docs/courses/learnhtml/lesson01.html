What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It defines the structure and content of a webpage that browsers read and display visually.

Think of building a house. First you build the skeleton — steel rods, concrete. HTML is that skeleton. CSS is the paint and decoration. JavaScript is the electricity and plumbing.

HTML File Basic Structure
<!DOCTYPE html>Must be the very first line — tells the browser this is HTML5
<html lang="en">Root element — everything lives inside this. lang matters for Google and screen readers
<head>Invisible metadata — title, charset, CSS links, SEO tags. Users don't see this directly
<body>All visible content — headings, paragraphs, images, buttons, everything users see
SEO Connection

Google's crawler reads your HTML to understand your content. Clean, semantic HTML = better crawl efficiency = better rankings. The lang attribute directly affects regional search rankings.

How Does a Browser Read HTML?

When you visit a website, here is what happens step by step:

  1. Your browser sends a request to the web server
  2. The server responds with an HTML file
  3. The browser reads top to bottom, line by line
  4. It renders a visual page — what you see on screen
Key Fact

HTML is a markup language — not a programming language. It has no logic, conditions, or loops. It only describes structure. For logic, you need JavaScript.

Common Mistake

HTML files must have the .html extension. Any text editor can write HTML — Notepad, VS Code, Sublime. The browser does the rendering, not the editor.

Why Does This Exist?

HTML made the internet possible. Tim Berners-Lee created the first website in 1991 at CERN using only HTML. Today, 1.9 billion websites all start with the same structure you are learning right now.
