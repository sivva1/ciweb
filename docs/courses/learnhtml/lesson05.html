Links and Anchors

The <a> tag creates hyperlinks - the reason the web is called a web. Without links websites are isolated islands. Links connect everything and power Google's entire ranking algorithm.

<a> TagAll Attributes
hrefDestination URL. Required. Can be absolute URL, relative path, or anchor #id
target="_blank"Opens in new tab. Always pair with rel="noopener noreferrer" for security
rel="noopener"Prevents new tab from accessing your page via window.opener. Security essential
href="#id"Jumps to element with matching id on same page. Used for table of contents
href="mailto:"Opens email client with pre-filled address when clicked
href="tel:"On mobile devices, tapping this calls the number directly
Three Link Types
External<a href="https://google.com"> - full URL with https://
Internal<a href="about.html"> - relative path to another page on your site
Anchor<a href="#section2"> - jumps to element with id="section2" on same page
Security Warning

Never use target="_blank" without rel="noopener noreferrer". Without it the new tab can access and modify your page using window.opener. This is a real attack used in the wild.

SEO: Link Text Matters

Never write "click here". Google reads anchor text to understand the linked page topic. Write "Read our HTML guide" instead of "click here".

Links Made Google Possible

Google's PageRank algorithm - which made Google a trillion dollar company - is entirely based on analyzing which pages link to which other pages. Tim Berners-Lee calls hyperlinks his greatest contribution to humanity.
